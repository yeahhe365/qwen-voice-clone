## Overview

A command-line and WebUI tool for Qwen voice cloning API workflows.

## Features

- Command-line voice cloning calls.
- WebUI mode.
- Supports authentication configuration.
- Outputs generated audio files.

## Quick Start

- Prepare a Python environment.
- Run `python qwen_voice_clone.py` to inspect CLI arguments.
- Run `python webui.py` to start the WebUI.

## Configuration

- Follow the README instructions to obtain and configure authentication data.

## Tech Stack

- Python
- Qwen voice service
- WebUI
